# Coospo-BK467-Receiver 
Connect to sensors and receive data via BLE

## Usage:
```c
./BK467-Receiver.exe
./BK467-Receiver.exe [IP] [PORT] # default: 0.0.0.0:10000
./BK467-Receiver.exe -h
```
